========================================
    Desktop Mate - Installation Guide
========================================

QUICK SETUP (2 Steps):
1. Install Node.js from https://nodejs.org
2. Double-click "install.bat"

That's it! Desktop Mate will be ready to use.

========================================

DETAILED INSTRUCTIONS:

Step 1: Install Node.js
- Go to https://nodejs.org
- Download the LTS version
- Install with default settings
- Restart your computer (recommended)

Step 2: Install Desktop Mate
- Double-click "install.bat"
- Wait for installation to complete
- Desktop shortcut will be created automatically

Step 3: Run Desktop Mate
- Look for "Desktop Mate.bat" on your desktop OR
- Double-click "run.bat" in the installer folder
- Your cute companion will appear!
