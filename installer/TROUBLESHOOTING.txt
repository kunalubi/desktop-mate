========================================
    TROUBLESHOOTING GUIDE
========================================

PROBLEM: Desktop shortcut not created?

SOLUTION 1: Use run.bat
- Look for "run.bat" in the installer folder
- Double-click "run.bat" to start Desktop Mate

SOLUTION 2: Manual shortcut creation
1. Right-click on desktop → New → Shortcut
2. Browse to installer folder
3. Select "run.bat"
4. Name it "Desktop Mate"
5. Click Finish

SOLUTION 3: Run directly
- Open installer folder
- Double-click "run.bat"
- Or open Command Prompt here and type: npx electron .

========================================

PROBLEM: "Node.js not found" error?

SOLUTION:
1. Go to https://nodejs.org
2. Download LTS version
3. Install with default settings
4. Restart computer
5. Run install.bat again

========================================

PROBLEM: Installation fails?

SOLUTION:
1. Check internet connection
2. Run Command Prompt as Administrator
3. Navigate to installer folder
4. Run: npm install electron
5. Then run: npx electron .

========================================

PROBLEM: Desktop Mate doesn't appear?

SOLUTION:
1. Check if it's running in background (Task Manager)
2. Try running as Administrator
3. Make sure no antivirus is blocking it
4. Restart computer and try again

========================================

QUICK START (If installer fails):
1. Open Command Prompt in installer folder
2. Type: npm install electron
3. Type: npx electron .
4. Desktop Mate should appear!
