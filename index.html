Installation Guide

1

Install Node.js

Download and install Node.js from nodejs.org

2

Download & Extract

Download the installer package and extract it to your desired location

3

Run Installer

Double-click install.bat and wait for installation to complete

4

Launch & Enjoy

Use run.bat or desktop shortcut to start your desktop companion!

Desktop Mate is a free, open-source virtual companion built with Electron. Bring life to your desktop with cute animations and interactive features.
