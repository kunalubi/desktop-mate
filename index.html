üìã Manual Setup Instructions

For Users (Simple):

  1. Download Node.js from nodejs.org
  2. Download the project files (ZIP)
  3. Extract to a folder
  4. Open Command Prompt in that folder
  5. Run: npm install
  6. Run: npm start
  7. Desktop Mate will appear on your screen!

For Distribution:

  1. Create a ZIP file with all project files
  2. Include a README with the above instructions
  3. Users extract and run npm install then npm start
